# Code of Conduct

This project adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct, unmodified.

The full text — including the Pledge, Standards, Enforcement Responsibilities, Scope, and Enforcement Guidelines — is the canonical version published at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

To report a concern, email **conduct@sureops.ai**. Reports are read by the sureops maintainers and handled per the Enforcement Guidelines in the Covenant.

For everyone's safety:

- Reports are confidential. We do not share reporter identity beyond the people directly involved in handling the report.
- We will not retaliate against anyone reporting in good faith.
- For security vulnerabilities (not conduct issues), use **security@sureops.ai** instead — see [SECURITY.md](./SECURITY.md).

## Scope

This Code applies to all sureops community spaces — this repository's issues, pull requests, discussions, and any official sureops events or channels (Slack, mailing lists, conferences) when present. It also applies to representations of the project in public spaces.
